Realtime Chat App

The Real-Time Chat Application is a cutting-edge full-stack solution built with Next.js 13, combining React for dynamic user interfaces and TailwindCSS for a responsive, modern design. The app features instant messaging powered by Upstash Redis, ensuring seamless real-time communication and high-performance data retrieval. Users benefit from a comprehensive friendship system that allows for easy management of friend requests, while Google Authentication secures user access. With TypeScript enhancing code reliability, this application showcases a sophisticated approach to real-time chat, emphasizing both performance and user experience.

Customer Support Chatbot

This innovative customer support chatbot is a full-stack solution built with Next.js, leveraging the power of OpenAI's ChatGPT for intelligent, real-time interactions. The chatbot offers instant, streamed responses to user queries, ensuring a seamless and engaging communication experience.

Security is a top priority, with Upstash providing rate-limited API routes to protect against abuse while maintaining high performance. The project also features optimistic UI updates, enhancing the overall user experience by making the interface more responsive. With robust error handling and TypeScript ensuring code reliability, this chatbot is a prime example of how cutting-edge technology can be applied to solve real-world customer service challenges.

    Maze Adventure is a first-person 3D maze exploration game developed using Three.js, a powerful JavaScript library for creating animated 3D graphics. The game features a detailed maze environment with textured floors and walls, dynamic lighting effects including ambient light, directional lights, and spotlights, all of which enhance the visual experience.     3D House

    This project involves creating a detailed 3D environment using Three.js, a powerful JavaScript library for rendering 3D graphics in the browser. The scene includes a dynamic water surface with realistic reflections and waves, created using the Water2.js class. A first-person perspective is implemented through a movable sphere that the camera follows closely. The environment also features a skydome with changing textures for day, sunset, and night, and a variety of 3D models including a house with detailed interiors, a floor, roof, furniture, and other objects imported using GLTFLoader. Lighting is carefully managed with ambient, directional, and spotlights to enhance realism. User interaction is enabled through keyboard controls, allowing movement and rotation of the first-person sphere, as well as changing the skydome phase, providing an immersive and interactive experience.

    VMs Allocation using Genetic Algorithm

    This project optimizes the allocation of Virtual Machines (VMs) to Cloud Service Providers (CSPs) using a genetic algorithm. Implemented in Java, it dynamically assigns 7 VMs across 8 CSPs by evaluating cost, reliability, and latency. The genetic algorithm iteratively refines the allocation strategy through population initialization, fitness evaluation, selection, crossover, and mutation. This approach balances multiple objectives to achieve an optimal deployment strategy, demonstrating significant improvements in cloud resource allocation efficiency.

    Space Invader

Players engage in interactive gameplay where they control a cannon to shoot lasers at alien adversaries.     Forest

    This project utilizes Three.js, a JavaScript library for creating 3D graphics in web browsers, to develop a dynamic scene featuring a lush environment. The scene includes a spherical skydome with a panoramic texture, providing a realistic backdrop. It incorporates a circular ground plane with a repeating grass texture, demonstrating texture mapping capabilities. The environment is populated with trees of varying types and sizes, generated procedurally and placed randomly within a defined circular area. Each tree type is distinct, featuring detailed trunks and foliage textured with images of branches and leaves. Lighting is employed to enhance the scene's visual depth and realism. The project showcases Three.js' ability to render complex 3D environments with interactive elements directly in the browser, offering a visually compelling experience for users
